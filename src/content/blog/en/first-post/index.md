## Introduction

In modern website development, especially blogs or documentation, the reading experience is very important. Using a neat layout, comfortable typography, and consistent navigation will greatly help readers.

This post aims to:
- Test the blog layout appearance
- Test typography (headings, paragraphs, lists)
- Test sticky features (sidebar, navbar, or TOC)

---

## What are Sticky Elements?

Sticky elements are UI elements that:
- Stay visible when the page is scrolled
- Usually used for:
  - Navbar
  - Category sidebar
  - Table of Contents (TOC)
